# SnapSecure AI - Frontend

The sleek, modern, and responsive user interface for **SnapSecure AI**, built with **React (Vite)** and **Vanilla CSS**.

##  Features
- **Dynamic Dashboard**: Interactive area for uploading and analyzing screenshots.
- **Real-time Feedback**: Visual indicators for analysis progress.
- **Blob-based Downloads**: Forced automatic downloads to bypass cross-origin browser restrictions.
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices.

##  Getting Started

### Prerequisites
- **Node.js**: v18+ recommended.
- **Backend Service**: Ensure the SnapSecure AI Backend is running on `http://localhost:8001`.

### Setup
1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

### Running the App
Start the Vite development server:
```bash
npm run dev
```
